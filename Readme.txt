MD code for the paper: Wei et al., Atomistic origin of montmorillonite clay subjected to freeze-thaw hysteresis, Journal of Rock Mechanics and Geotechnical Engineering, 2024. https://doi.org/10.1016/j.jrmge.2024.11.004.

01-cooling-270-230k.in: This input script was to calculate the cooling process from 270 K to 230 K, corresponding to Ice-water phase transformation process.  
02-heating-230-270k.in: This input script was to calculate the heating process from 230 K to 270 K.
03-pulling-230k.in: This input script was to calculate the SMD pulling, corresponding to Surface effect of clay to ice-water system.
